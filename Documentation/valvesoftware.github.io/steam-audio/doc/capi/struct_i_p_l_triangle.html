Detailed Description

A triangle in 3D space.

Triangles are specified by their three vertices, which are in turn specified using indices into a vertex array. See iplSetStaticMeshVertices for how to specify the vertex array. Phonon uses a counter-clockwise winding order. This means that when looking at the triangle such that the normal is pointing towards you, the vertices are specified in counter-clockwise order.
